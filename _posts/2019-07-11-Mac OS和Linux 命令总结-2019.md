# Mac OS/Linux命令查询网络端口占用情况
## netstat 命令

```

netstat -an | grep 50051

```

50051 可以替换成你需要查询的端口。

## lsof 命令
通过 list open file 命令可以查看到当前打开文件，在 linux 中所有事物都是以文件形式存在，包括网络连接即硬件设备。

```

lsof -i:50051

```

-i 参数表示网络连接，:80 指名端口号，该命令会同时列出 PID，方便 kill。
查看所有进程监听的端口：

```

sudo lsof -i -P | grep -i "listen"

```
